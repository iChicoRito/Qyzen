# Objective

## Student Assessment Score Data Expansion

---

## Description

Create a realistic data query for student assessment scores by expanding the current database, which currently contains scores for only one specific student. The goal is to include multiple students with different assessment scores.

---

## Objectives Breakdown

### 1. Main Objective Area

Develop a realistic data query structure for assessment scores across multiple students instead of limiting the data to a single student.

---

### 2. Secondary Objective Area

Expand the existing score database by adding data entries for every student with varying scores.

---

### 3. Supporting Tasks

#### 3.1 Task Group

* Create a realistic data query for assessment scores
* Expand the current database from one student to multiple students
* Add different assessment scores for each student

---

### 4. Detailed Breakdown

#### 4.1 Current Database Limitation

The current database contains assessment score data for only one specific student.

#### 4.2 Data Expansion Requirement

Additional student data must be added to the database.

#### 4.3 Score Variation Requirement

Each student should have different assessment scores.

##### Nested Details

* Include multiple students in the database
* Ensure scores vary between students
* Maintain realistic assessment score data only based on the provided objective
