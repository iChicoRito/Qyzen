  1). Now in the creation of the users, since we don't have any creation of password yet, I wanted that every user created, they're password is automatically created by using their combined given_name + surname e.g Given Name + Surname(Mark_Adrianne_Salunga).

    - Mark_Adrianne <- given_name, always put underscore everytime it has spaces
    - Salunga <- surname, always put underscore everytime it has spaces if ever.
